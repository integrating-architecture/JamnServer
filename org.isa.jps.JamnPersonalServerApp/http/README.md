# Jamn Web Workbench 

An example of a slim, independent Web UI e.g. for individual tools.

The UI is pure HTML/CSS and communicates via http/rest/websocket with the JamnPersonal Server App.

Individual user functions and services can be implemented on the app side in Java, Java-Script or Shell-Scripts<br>
and in HTML/CSS and Java-Script on the UI side, as required - without having to meet any specifications or any other restrictions.<br>
There is No deployment, No dependency management No external tooling required.
