# JamnServer
### Just Another Micro Node Server

Jamn is an **experimental**, lightweight socket-based server designed for independence, easy use and easy customization.
The purpose is textdata based interprocess communication e.g. with JSON/XML/HTML etc. based on a rudimentary HTTP compatibility.

The Server implementation has NO dependencies to external libraries, uses only standard Java SE classes and is implemented in ONE single class file.

Since the server supports HTTP only in a rudimentary, use case oriented way, it is **NOT SUITABLE FOR PRODUCTION PURPOSES** – but it is quick and easy to use for tooling, testing and concept experiments.

It is a Playground in change and progress ...
