# JamnWebRPCProvider

This is an experimental example implementation for simple, Rest-like RPC Web Services implemented as a JamnServer Provider PlugIn.

Required dependencies:

```xml
<dependencies>
	<dependency>
		<groupId>org.isa.ipc</groupId>
		<artifactId>org.isa.ipc.JamnServer</artifactId>
		<version>0.0.1-SNAPSHOT</version>
	</dependency>
	
	<!-- or or a tool of your choice -->
	<dependency>
		<groupId>com.fasterxml.jackson.core</groupId>
		<artifactId>jackson-databind</artifactId>
		<version>2.17.2</version>
	</dependency>
</dependencies>
```
