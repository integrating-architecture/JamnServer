# JamnWebContentProvider

This is a sample implementation of a simple file-based <a href="/org.isa.ipc.JamnWebContentProvider/src/main/java/org/isa/ipc/JamnWebContentProvider.java">Web Content Provider</a>.

This is NOT about reinventing the web server wheel.

The approach is to reduce the server requirements to basic networking capability and consider everything else as part of an individual application.
